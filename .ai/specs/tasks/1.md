# Phase 1 — Factory wired, panel opens

**Goal:** Alt+7 on a `.mk` file opens the Structure panel with no errors — even if it shows only an empty root.

**Flow:** Flow 1 — Build→Observe→Adjust

**References:** `design.md` → MkStructureViewFactory, plugin.xml addition

---

> **When a task is complete, mark it `[-]`.**

- [-] 1.1 Create `MkStructureViewFactory.kt` — implements `PsiStructureViewFactory`, returns a `TreeBasedStructureViewBuilder` whose `createStructureViewModel` returns a stub `StructureViewModelBase` with the `psiFile` as root and no children.

  **Test:**
  - Do: `./gradlew runIde`, open a `.mk` file, press Alt+7.
  - Expect: Structure panel opens and shows either an empty tree or a single unlabeled root node — no exception in the IDE log.

- [-] 1.2 Commit changes from 1.1.

  **Test:**

  - Expect: commit succeeds and `git status` shows a clean working tree.
