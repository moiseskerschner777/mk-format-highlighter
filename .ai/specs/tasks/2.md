# Phase 2 — Flat list of `:` lines

**Goal:** Structure panel shows all parent lines as a flat list; clicking any entry moves the caret to that line.

**Flow:** Flow 1 — Build→Observe→Adjust

**References:** `design.md` → MkStructureViewElement, MkStructureViewTreeModel, Data flow

- [ ] 2.3 Create `MkStructureViewTreeModel.kt` — implements `StructureViewModelBase`. `buildRoot()` scans `psiFile.text` line by line, collects every line where `line.trimEnd().endsWith(":")`, computes `depth = min(leadingSpaces / 4, 6)` and `offset` (accumulated char count). Returns a root `MkStructureViewElement` whose `children` is the flat list of all collected nodes (all depths, no nesting yet). Update `MkStructureViewFactory` to use `MkStructureViewTreeModel` instead of the stub.

  **Test:**

  - Expect: Structure panel shows every `:` line as a flat entry. Labels match the line text without the trailing `:`. Plain lines do not appear.
